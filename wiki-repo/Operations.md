# Included Formatters & Operations

`ts-chef` provides a wide array of professional-grade tools for data formatting and transformation.

## Beautifiers

-   **JSON Beautify:** Formats JSON data with consistent indentation.
-   **XML Beautify:** Adds indentation and line breaks between XML tags.
-   **CSS Beautify:** Formats CSS rules for better readability.
-   **SQL Beautify:** Formats complex SQL queries (SELECT, INSERT, etc.).
-   **JavaScript Beautify:** Formats JS/TS code using standard styles.
-   **Generic Code Beautify:** A fallback beautifier for various other languages.

## Minifiers

-   **JSON Minify:** Removes all unnecessary whitespace/comments from JSON.
-   **CSS Minify:** Compacts CSS to reduce file size.
-   **SQL Minify:** Compacts SQL queries.
-   **JavaScript Minify:** Minifies JS code.

## Utility Operations

-   **Format MAC Addresses:** Standardize MAC address formats (colon, dash, none).
-   **Add Line Numbers:** Prefixes each line with its index.
-   **Remove Line Numbers:** Cleans up text by removing leading numbers.
-   **Sort:** Sorts lines based on various criteria (alphabetic, numeric, etc.).
-   **Unique:** Removes duplicate lines from a list.
