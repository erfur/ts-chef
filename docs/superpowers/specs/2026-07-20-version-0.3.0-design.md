# Version 0.3.0 Design

## Goal

Release the pending README logo restoration as version `0.3.0`.

## Design

Update the project version in `package.json` and the root package metadata in `package-lock.json` from `0.2.0` to `0.3.0` without changing dependency versions. Use npm's version command without creating a commit or Git tag.

## Verification

Confirm both package metadata files report `0.3.0`, package the extension as `vschef-0.3.0.vsix`, and inspect the packaged manifest to confirm version `0.3.0`.
